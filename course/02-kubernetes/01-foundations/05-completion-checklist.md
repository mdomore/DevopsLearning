# Phase 1 completion checklist

Mark Phase 1 done when you can:

- Identify nodes + Pod placement (basic scheduling observation)
- Use `kubectl get`, `kubectl describe`, `kubectl logs`, `kubectl exec`
- Write a minimal YAML Pod and update it with `kubectl apply`
- Diagnose at least two failures using `describe` + `events` + `logs`
